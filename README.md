## 🥂 Overview

**Flourish** is a premium e-commerce web application designed for a luxury, artisanal bakery based in the heart of Islamabad. The platform serves as a digital storefront for high-end pastries, bespoke wedding cakes, and seasonal indulgences.

## 🤍 Key Features

*   **Editorial Product Galleries:** Magazine-style, masonry layouts.
*   **Bespoke Cake Pre-orders:** Seamless user flow for custom orders.
*   **Dynamic Shopping Cart:** Sliding side-cart UI with real-time subtotal calculations.
*   **Fully Responsive Design:** Optimized for MacBook, iPad, and mobile devices.

---

## 💻 Tech Stack

*   **Frontend:** React.js / Next.js
*   **Styling:** Tailwind CSS
*   **State Management:** Redux / React Context API
*   **Database/Backend:** Firebase / Node.js
*   **Design & Prototyping:** Figma
